wordpress对接微信小程序

发表于2021-06-19|更新于2023-01-25|other教程

不做开发，但系统运维偶尔需要和开发打交道，那就折腾吧。

wordpress对接微信小程序，百度+视频用了1小时左右搞定，用“微慕”和“丸子”的开源小程序源码做了测试，当然还有其它的。两个字“简单”，哈哈哈，原谅我这个门外汉的大言不惭，毕竟我只是入门级的。

简单讲就是：小程序源码+wordpress接口插件

微慕vs丸子：

微慕：简单，简单，什么都简单，功能也简单，毕竟玩的开源的，想玩复杂的可以去微慕官网买专业版的，呵呵呵呵

丸子：功能多了一点，评论、点赞、审核通知、订阅，也就预示着自己要修改的地方有点多；小程序要调整页面效果，不调整也可以用，就是别人的效果永远不是自己理想的效果。

最终我还是使用了丸子小程序源码，搞了一天半小程序审核通过，真的要感谢那些开源的大佬和团队，这里顺便记录一下搭建过程。至于微慕的网上教程很多，视频也很多，自己墨迹一下也可以搞定的。

贴上我的小程序：

先决条件：

  • 域名已备案
  • 域名已配置ssl（https）
  • 已搭建wordpress 4.7 以上（4.7 版本以上的源码包含 REST API JSON 接口）
  • 微信小程序开发者账号（个人、企业都可以）

如果以上条件没有满足就先去折腾上面的内容，满足条件继续下个步骤。

下载工具及软件包

微信开发者工具：官方下载

丸子wordpress插件：wordpress插件商店下载、或 github下载

丸子小程序源码：github下载

我修改后的微信小程序源码：gitee下载

流程

  • 小程序后台设置
  • wordpress后台设置
  • 小程序源码配置测试
  • 上传小程序源码，并发布小程序

一，小程序后台设置

小程序后台：获取appid和秘钥；设置小程序服务器域名地址。

扫码登录小程序后台：https://mp.weixin.qq.com/

1.获取appid和秘钥：开发–>开发管理–>开发设置，

我的appid：wxbec70195f60f5f12

秘钥：第一次会提示“生成”，切记保存秘钥（服务器不保存秘钥）

将appid和秘钥保存一下一会要用到！

2.生成完秘钥，往下滑动，就可以看到服务器域名设置，全部写上你的wordpress地址。

注意：downloadfile地址必须加入腾讯的两个地址，提供微信头像https://thirdwx.qlogo.cn；https://wx.qlogo.cn

二，wordpress后台设置

1.安装丸子wordpress插件

插件–>安装插件–>关键字“mini Program API”–>安装–>启用后左上角就可以到“小程序”模块了

2.配置“丸子小程序”插件

aaa.小程序授权，输入获取到的appid和秘钥

bbb.常规设置，我只设置了首图显示的小程序名称和描述

ccc.功能扩展，按需勾选，我只是测试什么都没勾选。

ddd.小程序评论开启审核（小程序审核建议开启）

三，小程序源码配置测试

导入小程序源码绑定自己的appid，并修改自己的url地址。

1.打开微信小程序开发工具，

2.导入下载的微信小程序源码

3.输入上面获取到的自己的appid，并选择本地后，“新建”完成。

4.修改自己的url地址

5.修改完成后，“清除全部缓存”—>刷新”编译”，即可看到自己小程序效果。

6.修改源码，如果对代码很熟悉可以直接修改，对于我这种不是很熟悉的人，喜欢使用“可视化”进行修改，当然也需要一定的基础，否则容易把源码搞废。

四，上传小程序源码

当本地测试完美后，我们可以上传小程序源码，并在小程序后台发布了。

1.开发者工具，上传源码。第一次上传默认即可。

2.小程序后台，提交版本审核申请

3. 关于提交审核的建议

aaaa.修改自己小程序类目，尽量以 教育 > 教育信息 和 服务工具 > 信息查询

bbb.关于版本说明尽量体现小程序的功能，不要出现测试字眼。

ccc.我申请时写的：wordpress对接小程序；文章实时更新、评论、审核；必应图库api
