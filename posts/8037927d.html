centos7-安装配置phpmyadmin

发表于2021-03-27|更新于2023-01-25|Linux

phpMyAdmin是一个非常受欢迎的基于web的MySQL数据库管理工具。它能够创建和删除数据库，创建/删除/修改表格,删除/编辑/新增字段，执行SQL脚本等。

  • 一，安装lnmp（nginx+php+mysql）
  • 二，下载phpmyadmin安装包，并解压到nginx站点目录
  • 三，nginx配置虚拟主机后
  • 四，防火墙放行端口，关闭selinux
  • 五，重启nginx，测试phpmyadmin站点

第一步，安装nginx（web服务器）

#安装wget(下载工具)
yum -y install wget
#安装unzip（zip解压工具）
yum -y install unzip
#安装lnmp（下面我个人的gitee脚本，如某个rpm已安装会自动跳过）
curl -o lnmp-1.5.7_init.sh https://gitee.com/ffing/onekey/raw/master/lnmp-1.5.7\_init.sh && bash lnmp-1.5.7_init.sh

第二步，下载phpmyadmin安装包，并解压到nginx站点目录

phpmyadmin官网：https://www.phpmyadmin.net

第三步，nginx配置虚拟主机后，重启nginx
